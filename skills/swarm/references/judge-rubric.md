# Structured Judge Rubric

Every synthesis/merge step must produce these five sections **before** the final answer:

1. **Consensus** — points all or most agents agree on (treat as higher-confidence).
2. **Contradictions** — where agents disagree; name the conflict and adjudicate.
3. **Partial coverage** — points only some agents reached.
4. **Unique insights** — valuable points from a single agent.
5. **Blind spots** — what none of them addressed but the task needs.

Then write the **grounded final answer**, citing sources/agents inline. Never invent
consensus that isn't there; if confidence is low, say so.
